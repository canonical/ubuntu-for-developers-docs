Project and community
---------------------

* |Read our code of conduct|_: As a community we adhere to the 
  Ubuntu code of conduct.
* |Get support|_: Ask Ubuntu is a question and answer site
  for Ubuntu users and developers.
* |Join our online chat|_: Meet us in `#ubuntu` on IRC Libera.Chat.
* |Report bugs|_: We want to know about the problems so we can fix them.
* |Contribute code|_: The code is open and we are open
  to accepting changes to it.

Thinking about using Ubuntu Desktop as your development platform?
`Get it! <https://ubuntu.com/desktop/developers>`_
